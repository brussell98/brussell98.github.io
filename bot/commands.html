Standard

Command Description Example Aliases
@Bot-chan Text Doing this will let you talk to the bot.
]help [command] Get a list of commands or information on a specific command. h, commands
]id [channel] Get your user ID. If the suffix is "channel" get the channel's ID. myid
]ping Check if the bot is working. Tells you response time in ms. p
]beep boop
]invite Get the link to add the bot to a server. join
]about Display info about the bot.
]dice [(rolls)d(sides)] Roll a dice with 6 sides or as specified.
]roll [max] Roll a random number between 1-max or 1-100.
]info [user] Get info on the server, or, if specified, a user i, user, userinfo, serverinfo
]choose <option> | <option> | [option] ... Makes a choice for you. ]choose sleep | discord c, pick
]lotto end | enter | new [max entries] [-noautoend] | <pick from> | everyone Start / stop / enter a lottery. Allows setting max entries per user, auto end, predefined entries, and picking from all members. ]lotto new 2 -noautoend ]lotto @Brussell @Not-Brussell
]strawpoll <option> | <option> | [option] ... Create a strawpoll. ]strawpoll DOTA2 | LoL straw, poll
]vote new [topic] | +/- | end Vote on something. ]vote new Ban @GenericUsername v
]8ball [question] Ask the magic 8 ball. ]8ball 8, ball
]anime <search> Get info on an anime from MAL. ]anime Boku Dake animu
]manga <search> Get info on a manga from MAL. ]manga hero academia mango
]maluser <user> Get info on a user's MAL. ]maluser brussell98 mal
]coinflip Flip a coin. ]coinflip coin, flip
]avatar [user] Get a user's avatar. ]avatar Bruss a
]rps Rock, paper, scissors. ]rps
]weather <city | zip code> [,country code] Get the weather for a place. Uses Open Weather Map. ]weather new york,us w
]botserver Get an invite to the Bot-chan server. ]botserver server
]numberfacts [number] Facts on a number. ]numberfacts 42 num, number
]catfacts Catfacts! ]catfacts cat, meow, neko
]ratewaifu [--s[earch]] Rate a waifu. Works on users too! ]ratewaifu Rikka r, rate, waifu
]image <subreddit> [--nsfw] [--day | --week | --month | --year | --all] Get a random image from Imgur. ]image awwnime --month imgur
]osu ... Check out this fancy page for the osu! commands
]fortune [category] Get a fortune from yerkee.com/api. The categories are: all, computers, cookie, definitions, miscellaneous, people, platitudes, politics, science, and wisdom. By default wisdom is selected. ]fortune cookie f

]mentions Find out why you were mentioned. ]mentions
